Multus Helm Repository

This GitHub Pages site hosts the multus-quickstart Helm chart. You can add it as a Helm repository and install it into your Kubernetes cluster.

Repository URL

Helm repository URL: https://rohank-kz.github.io/Multus

Usage

From your terminal:

helm repo add multus https://rohank-kz.github.io/Multus
helm repo update
helm install multus multus/multus-quickstart -n kube-system --create-namespace

About the Chart

  • Installs Multus CNI using the upstream quickstart-style DaemonSet.
  • Includes the NetworkAttachmentDefinition CRD and required RBAC.
  • Defaults are tuned for quick testing; review before production use.

Chart source and issues: github.com/rohank-kz/Multus
